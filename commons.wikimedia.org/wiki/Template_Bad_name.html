Usage

{{Bad name |1= }}

You can also simply use the name of the correct category as parameter. See the example below.

Template parameters

ParameterDescriptionDefaultStatus
1Correct nameName of the correct categoryemptyrequired

Additional information

The template is intended to be used in the following namespaces: the Category namespace

The template is intended to be used by the following user groups: no user group specified

Placement: At the top of the page

Example

{{Bad name|Category:Unidentified‎}}

renders as:

This page has an incorrect name.
A correctly named duplicate category exists at:
Category:Unidentified‎

This page qualifies for speedy deletion.

  • Please make sure this template is on the page that should be deleted – inversions are frequent.
  • Please use {{Bad name|Correctly named category}} and make sure the correctly named category appears above.
For administrators:
Follow the directions on Commons:Deletion policy
  1. Add to CommonsDelinker's commands:
    {{Move cat|Bad name|Category:Unidentified‎|reason=incorrectly named duplicate}}
  2. Delete this page.

To add your language to the list of languages supported by this template, please copy the code of Template:Bad name/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box).
