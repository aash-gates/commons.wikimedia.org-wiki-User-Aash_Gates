First steps tutorial

Our first steps tour and our frequently asked questions will help you a lot after registration. They explain how to customize the interface (for example the language), how to upload files and our basic licensing policy (Wikimedia Commons only accepts free content).

You don't need technical skills in order to contribute here. Be bold when contributing and assume good faith when interacting with others. This is a wiki.

Getting help

More information is available at the community portal. You may ask questions at the help desk, village pump or on IRC channel #wikimedia-commons (webchat). You can also contact an administrator on their talk page. If you have a specific copyright question, ask at the copyright village pump.

Goodies, tips, and tricks
  • Put Babel boxes on your user page so others know what languages you can speak and indicate your graphic abilities.
  • You can find the files you have uploaded in your Uploads.
  • Please sign your name on Talk pages by typing ~~~~.
  • To link to an image page without embedding the image, type: [[:File:Foo.jpg]], which produces: File:Foo.jpg.
  • You may enable gadgets (custom features) for your account.
  • There are several tools to upload files.
  • Have you edited Wikipedia before? This guide might be useful.
Made a mistake?
  • Do you want to have a file renamed or moved? Use the move link and wait for an authorized user to rename it.
  • Do you want to have your recently uploaded picture removed? Tag it as {{speedy|reason here}}. For more information read the deletion guidelines.

Hello, Liuxinyu970226!

Thanks a lot for contributing to the Wikimedia Commons! Here's a tip to make your uploads more useful: Why not add some categories to describe them? This will help more people to find and use them.

Here's how:

1) If you're using the UploadWizard, you can add categories to each file when you describe it. Just click "more options" for the file and add the categories which make sense:

2) You can also pick the file from your list of uploads, edit the file description page, and manually add the category code at the end of the page.

[[Category:Category name]]

For example, if you are uploading a diagram showing the orbits of comets, you add the following code:

[[Category:Astronomical diagrams]]
[[Category:Comets]]

This will make the diagram show up in the categories "Astronomical diagrams" and "Comets".

When picking categories, try to choose a specific category ("Astronomical diagrams") over a generic one ("Illustrations").

Thanks again for your uploads! More information about categorization can be found in Commons:Categories, and don't hesitate to leave a note on the help desk.

Berne Convention date corrections[edit]

I see you are working through COM:CRT articles correcting information on accession to the Berne convention. This is good. Sometimes WIPO is not consistent. E.g. for the Cook Islands [1] shows the date as August 2, 2017 and [2] shows the date as August 3, 2017. I suggest that it would be best to add a citation to the source with the correct date, rather than change the date but leave the citation to the source with the incorrect date, which is confusing.

To make life easier for translators, the citation details can be hidden using <tvar|ref1> ... </>. Thus:

<ref>{{cite web |url=https://www.wipo.int/treaties/en/notifications/berne/treaty_berne_260.html |accessdate=2020-03-25 |title=Berne Notification No. 260 |publisher=WIPO |date=31 October 2013}}</ref>

On pages in languages other than English the translator will just see $ref1 in place of the citation details. Good work! Aymatth2 (talk) 12:46, 25 March 2020 (UTC)Reply[reply]

Thx. --Liuxinyu970226 (talk) 12:47, 25 March 2020 (UTC)Reply[reply]
