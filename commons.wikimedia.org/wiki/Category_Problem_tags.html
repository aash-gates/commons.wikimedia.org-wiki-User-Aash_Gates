Category:Problem tags

From Wikimedia Commons, the free media repository
Jump to navigation Jump to search
This category is used for categorizing templates.
It should not contain media files or pages outside the template namespace.

This is a tag category that should contain all templates that mark a serious problem with an image. This includes requests for deletions, license problems, as well as technical issues. It is intended for use by external tools and MediaWiki extensions, to easily identify tags that mark problems.

To assist in translating these tags, please see the talk page.
