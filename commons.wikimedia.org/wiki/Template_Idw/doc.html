Usage

{{subst:Idw |1= |2= |3= }}

Template parameters

ParameterDescriptionDefaultStatus
1the filename (with namespace such as File:Example.jpg)emptyrequired
2the pagename under "Commons:Deletion requests/"emptyoptional
3pluralizes ("3=plural" should work).emptyoptional

Additional information

The template is intended to be used in the following namespaces: the User talk namespace

The template is intended to be used by the following user groups: all users

This template is localized through {{Autotranslate}}. The layout of the template can be found under Template:Idw/layout.

To add your language to the list of languages supported by this template, please copy the code of Template:Idw/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.


The headline of this template will show up in different language if the file for deletion is not specified. To create a new language version of the headline of this template, create a new subpage (example: Template:Idw/heading/en) and use this code: 

{{#if:{{{1|}}}|[[:{{{1}}}]]|File deletion warning}}<noinclude>
{{Translated tag|user}}
</noinclude>
