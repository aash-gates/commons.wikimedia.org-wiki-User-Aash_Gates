This documentation is transcluded from Template:Technically replaced/doc.

When an image (usually an SVG file) has a rendering bug, and you want it to be replaced by another one for technical reasons, you may use this template. The librsvg used by MediaWiki has bugs rendering some SVG images, for such cases, you may export and upload a good PNG version, then use this template to let users use the PNG version when they include the image in a Wikipedia or other Wikimedia project site's page.

Template parameters

ParameterDescriptionDefaultStatus
1The image as the primary replacement for the original one.emptyrequired
descriptionDescription about the rendering bug, e.g. which part of this image has which rendering bug.emptyoptional
subsubcategory name to indicate the software producing the SVG file, e.g. :Inkscape to categorize into Pictures showing a librsvg bug:Inkscape. Currently allowed names: (use : as prefix) Adobe, Batik, Feedbin, HandSVG and Inkscape
Note: Use sub=- to suppress any categorization		emptyoptional

Localization

This template is localized through {{Autotranslate}}. The layout of the template can be found under Template:Technically replaced/layout.

To add your language to the list of languages supported by this template, please copy the code of Template:Technically replaced/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.
