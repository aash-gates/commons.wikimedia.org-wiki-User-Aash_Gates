This page offers language templates for galleries, categories description, file description pages to write text with language label before it. For other localization features see Commons:Localization.

You can simply use the ISO 639 code of your language (the same as the subdomain of the Wikipedia in your language, so en for English, de for German etc.) as template name. The first and only parameter of the template must be the whole text that is in the specific language.

Sample description section
Wikitext:

{{de|Eine [[:de:Beschreibung|]] ist ein Text, der die Eigenschaften eines Objekts spezifiziert.}}
{{en|A [[:en:description|]] is a text that specifies the properties of an object.}}

This renders as:
Deutsch: Eine Beschreibung ist ein Text, der die Eigenschaften eines Texts spezifiziert.
English: A description is a text that specifies the properties of an object.

Using templates instead of manual captions (i.e. writing '''en:''') is preferred because it is possible to use CSS to "hide" languages a person doesn't understand. This can reduce the amount of unnecessary information and visual clutter.

If the description includes a "=", you need to use

{{en|1=description "1+1=2" }}

or

{{en|description "1+1{{=}}2" }}

instead of

{{en|description "1+1=2" }}
