  • See subcategories for a breakdown of all candidates:
    • Category:Copyright violations — tag with {{copyvio|optional URL}}
    • Category:Duplicate — tag with {{duplicate|other file}} or {{Rename|good file name}}
    • Category:Advertisements for speedy deletion (spam) — tag with {{SD|G10|optional explanation}}
    • Category:Personal files for speedy deletion — tag with {{SD|F10|optional explanation}}
    • Category:Other speedy deletions — tag with {{speedydelete|reason}}
Note: {{Speedydelete}} is deprecated for most uses (files tagged with it get auto-categorized to Category:Other speedy deletions instead).
  • Before adding or deleting any file here, you must understand the speedy deletion policy.
