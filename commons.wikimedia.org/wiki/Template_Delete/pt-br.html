Este(a) predefinição foi nomeado(a) para eliminação.
Para discuti-la, visite a página de nominação.

Não remova esta etiqueta até que o pedido de eliminação seja fechado.

Razão para o pedido de eliminação: Nenhuma razão fornecida. POR FAVOR, forneça uma razão, ou este item terminará em Category:Incomplete deletion requests.
Clique aqui para mostrar mais informações

Se esta predefinição foi adicionada por você ter clicado em “Mark for deletion” no menu à esquerda, certifique-se de que todas as páginas necessárias tenham sido criadas. Se elas não foram criadas depois de poucos minutos, ou se você adicionou esta etiqueta manualmente, você deverá completar estes passos:

  1. Visite a subpágina dos pedidos de eliminação e coloque o seguinte código:
    {{subst:delete2|image=Template:Delete/pt-br|reason=}} ~~~~
  2. Vá para o registro de pedidos de eliminação e coloque o seguinte código no final:
    {{subst:delete3|pg=Template:Delete/pt-br}}
  3. Notifique o caso ao carregador ou criador da página colocando o seguinte código na página de discussão dele:
    {{subst:idw|Template:Delete/pt-br}} ~~~~
  4. Adicionalmente, você pode querer checar os projetos da Wikimedia que usam este item e então remover ou, se possível, trocá-lo por um item superior.

Para eliminações em massa: Se você quiser nomear vários arquivos relacionados, faça um pedido em massa colocando manualmente {{delete|reason=a razão|subpage=o nome|year=2022|month=October|day=15}} em cada arquivo ou página e então siga os passos acima. (Ajuda no pedido de eliminação em massa.)

Nota: esta predefinição é para pedidos que possam requirir discussão para que se chegue a um consenso. Para eliminações rápidas, você pode usar {{speedy|razão}}.
