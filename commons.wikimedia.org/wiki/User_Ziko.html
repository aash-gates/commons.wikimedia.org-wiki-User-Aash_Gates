Hello, I am Ziko van Dijk. On Wikimedia Commons, I mainly upload pictures that I took myself. I also like Wikimedia Commons to search for images that I embed in Wikipedia articles. Find more information about me on my page on Meta Wiki.

About my user name: before the introduction of the Single User Login for all Wikimedia wikis, my user name here on Wikimedia Commons was Ziko-C. If you want to use one of my media files, please write Ziko van Dijk for the name attribution.
