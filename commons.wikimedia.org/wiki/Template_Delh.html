  • Template:DeletionHeader/styles.css

Provides the header for the top of an archived deletion request. Must be used together with {{DeletionFooter}}.

Usage

{{DeletionHeader}}

Or use the shorthand {{Delh}}

The template takes no parameters.

Additional information

The template is intended to be used in the following namespaces: the Commons namespace

The template is intended to be used by the following user groups: all users

Placement:

just above the header
