This template marks all images released under PD-algorithm (any work done entirely by an artificial intelligence or computer algorithm).

The template automatically sets the following categories:

  • Category:PD-algorithm

Usage

{{PD-algorithm}}

The template takes no parameters.

Additional information

The template is intended to be used in the following namespaces: the File namespace

The template is intended to be used by the following user groups: all users

Placement: In the "Permission" parameter of the {{Information}} template (or the respective parameter of similar templates) or in the "Licensing" section

Relies on:
{{PD-Layout}}

Localization

Deutsch ∙ English ∙ français ∙ galego ∙ Nederlands ∙ العربية ∙ 中文（简体） ∙ македонски ∙

This template is localized through {{Autotranslate}}. The layout of the template can be found under Template:PD-algorithm/layout.

To add your language to the list of languages supported by this template, please copy the code of Template:PD-algorithm/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.
