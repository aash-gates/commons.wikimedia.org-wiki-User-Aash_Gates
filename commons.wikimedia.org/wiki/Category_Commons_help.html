This category contains help and tips on contributing to Commons.

  • Only help pages specific to Wikimedia Commons will be kept in Wikimedia Commons. All pages about the MediaWiki software itself will be removed in Wikimedia Commons as maintaining a full software handbook are duplicate efforts.
  • Thus all documentation on how to use MediaWiki syntax and such is hosted off-site at Meta-Wiki.
