Helpful info[edit]

  • Sample search string: insource:/\[\[Category\:Categories requiring permanent diffusion to zero/
  • Use {{Badname|Goodname}} to delete duplicate categories
  • File moving:
    • Use Move tab to move image
    • Add command such as {{universal replace|old name.jpg|new name.jpg|reason=[[Commons:File renaming|File renamed]]: Change from completely meaningless names into suitable names, according to what the image displays.}} to User:CommonsDelinker/commands/filemovers
  • Category renames (e.g., if target exists as redirect):
    • User talk:CommonsDelinker/commands/Category moves
  • Asking a user to use subcats
