This category contains files that are not permissible by the Commons licensing policy. As such, these files are subject to speedy deletion.
  • Admins can use the Bad Old Ones tool to assist deletion. (Not mandatory)
  • Check whether a file has been previously uploaded to Wikimedia Commons.
  • If a file is not an obvious speedy deletion candidate, please move it to normal Commons:Deletion requests. That will be faster for everyone.
  • Before adding or deleting any file here, you must understand the speedy deletion policy. Please read it in your language of choice:
