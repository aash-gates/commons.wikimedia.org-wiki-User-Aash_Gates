Template documentation[ view · edit · history · purge ]
This documentation is transcluded from Template:Disambig/doc.

This template marks all category pages that serve as a disambiguation page.
It adds tagged pages to Category:Disambiguation and additionally to Category:Non-empty disambiguation categories if they contain items.

Usage

{{Disambig}}

Generally this template would be placed following the list of items to be disambiguated (at the end of the article page).

The template takes no parameters.

Additional information

The template is intended to be used in the following namespaces: the Category namespace

The template is intended to be used by the following user groups: all users

This template is localized through {{Autotranslate}}. The layout of the template can be found under Template:Disambig/layout.

To add your language to the list of languages supported by this template, please copy the code of Template:Disambig/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.
