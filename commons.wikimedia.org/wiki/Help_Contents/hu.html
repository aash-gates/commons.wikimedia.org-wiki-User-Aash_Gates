A Wikimédia Commons szabad képek, videók, hangfelvételek és egyéb multimédiafájlok gyűjteménye. A feltöltött fájlok helyi fájlokként használhatók más Wikimédia-projektekben, így például a Meta-Wikiben, a MediaWiki.org-on, a Wikikönyvekben, Wikihírekben, Wikipédián, Wikidézetben, Wikiforrásban, Wikiegyetemen, Wikivoyage-ban és Wikiszótárban, vagy InstantCommons funkcióval akár nem Wikimédia-wikiken is.

This page is an index of all Wikimedia Commons' maintenance and Help pages. The following articles contain guidance and information about reading, authoring, and participating in the Commons community.

Nem találod, amit keresel? Ha nem szerepel a gyakran ismételt kérdések között, kérhetsz segítséget a magyar kocsmafalon vagy az angol help desken.

Segítség kérése:

  • If you want to post a question about copyright issues that another editor will respond to, you can go here.
  • If you have a general question about editing, uploading, the scope of the Wikimedia project, or anything else, you can post a question at the Help Desk and another user will reply.

MediaWiki

The function of MediaWiki itself – the software that Wikimedia Commons runs on – is described at the MediaWiki User's Guide in Meta-Wiki. If you suspect a software bug, ask for feedback at the village pump, and then report it using the Phabricator bug report system. This is the only way to get MediaWiki developers to notice your bug reports.

The database of Wikimedia Commons can be downloaded. There is currently no dump of the media files available.
