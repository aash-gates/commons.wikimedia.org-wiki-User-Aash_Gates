Featured media - Wikimedia Commons

Featured media are what we believe to be the best media files on Commons. Prior to being listed here, files are reviewed at Commons:Featured media candidates.

There are currently 232 such media in the Commons repository which is roughly 0.0003% of the available files (87,583,846).

Simply click on the "more buttons" of a gallery on this page to see all the featured media for that subject. To reuse any file from Wikimedia Commons please see this guide.


For the latest featured media, see this month's featured log. For a chronological list of featured media visit Commons:Featured media/chronological.

See Commons:Media help for help accessing a file.
