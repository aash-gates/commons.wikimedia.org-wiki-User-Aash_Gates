English: This is the top-level category for Commons-related administration and maintenance files, and these namespaces: Commons:, Template:, and Help:

Public content pages, such as media files and image galleries, should go elsewhere. The main category for public contents on Commons is Category:Topics. The main category for our tree data structure is Category:Categories.

The very topmost category over everything is Category:CommonsRoot.

Translated project pages in specific languages should be categorized under Category:Commons by language.

The subcategory Category:Commons maintenance content is for special maintenance tasks of Wikimedia Commons; it does not get translated.
This category should only contain other categories. Galleries should be in the corresponding categories instead. Files should be in a parent or subcategory.
