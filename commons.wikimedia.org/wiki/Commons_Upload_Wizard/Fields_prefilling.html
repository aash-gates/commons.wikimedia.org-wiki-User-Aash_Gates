It is possible to provide to the UploadWizard parameters in the URL to set default values for some of the upload fields

The fields that can be passed to the Upload Wizard are:

caption
The caption to be used for the image (caption is a term of Structured Data)
captionlang
To use a different caption language than the default one; uses a 2 or 3 letters language code, e.g. en for English or fur for Friulian.
description
The description to be used for the image
descriptionlang
see captionlang above
categories
List of categories, separated with pipes « category1|category2 »
lat
Latitude of the image (interpreted as camera location, not object location)
lon
Longitude of the image (interpreted as camera location, not object location)
alt
Altitude of the image
campaign
The name of the Upload Campaign
fields[]
Passing parameters to the upload campaign
id
The id field. This is for upload campaigns where it is possible to pass an id (deprecated)
