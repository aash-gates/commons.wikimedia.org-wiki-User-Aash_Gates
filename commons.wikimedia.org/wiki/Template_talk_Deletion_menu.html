Suggestion for improvement of template[edit]

Hello! I've just spent the last three hours working on a prototype mockup that would modify this template to show how many members are in each category. First, I had to create a parser to parse the category out of the template, see Template:Deletion menu/layout/parser. Then, using that parser, I modified a sandbox copy of the layout subpage to return produce the desired output, see Template:Deletion menu/layout/sandbox. Then, as a testcase so-to-speak I used a sandbox copy of the /en version of the template to see what the end result would look like with the fabricated piped links that are used here, see Template:Deletion menu/en/sandbox. I think it is a great improvement to see what is in which category and it show which cats are empty. It is currently set up to only count the number of FILES in the category, but this could easily be altered to count anything. I need some feedback on what should be counted in each, and I would be happy to make the modifications and implement them if there is consensus. I would also be willing to completely redesign the template to use a table similar to what is used on enwp in en:w:Template:Admin dashboard/csd. — {{U|Technical 13}} (e • t • c) 17:43, 30 November 2014 (UTC)Reply[reply]
