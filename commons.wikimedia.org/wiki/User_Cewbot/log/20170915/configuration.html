Template:Bot use warning 本頁面為 cewbot 增加討論頁面主題列表作業的設定。每次執行作業前，機器人都會從本頁面讀入設定。本設定頁面將影響主題列表功能，應受適當保護以免受破壞。且應謹慎編輯，以防機器人無法讀取。移動本頁面必須留下重定向。

  • 請注意：變更本頁面後，起碼必須等數分鐘，機器人才會應用新的設定。
  • 原始碼 (source code) on GitHub
  • 已知無法解決問題：目前 MediaWiki 之 link anchor, display_text 尚無法接受"�"這個特殊字元。

一般設定[edit]

general
變數名 (不可更改) 變數值 註解說明
stop_working false 停止運作 (請填上true/false才能正常運作)。
max_title_length 40 當標題過長，大於 max_title_length 時，縮小標題字型。
max_title_display_width 24em 限制標題欄的寬度（要考慮螢幕較窄時情況）。

討論議題列表顏色[edit]

list_style
變數名 (不可更改) 顏色 註解說明
1h #efe 最近一小時內背景：淺綠色。
1d #eef 超過一小時到最近一日內背景：淺藍色。
1w #ddd 超過一禮拜到一個月背景：淺灰色。
1 month #bbb 超過一個月背景：深灰色。

已移動或結案的議題之顯示格式[edit]

closed_style
變數名 (不可更改) 設定 / 格式 註解說明
show_subtopic false 顯示次級連結 (請填上true/false才能正常運作)。
line_CSS text-decoration: line-through 整行顯示格式。不包含連結文字顏色。
link_CSS 連結文字顯示格式。
link_color #111 連結文字顏色。
link_backgroundColor 連結文字背景顏色。

顯示主題列表之頁面[edit]

若有需求，您可自行於下列表中增加頁面，但必須先手動造出"/topic list"這個子頁面。機器人常時執行。本段設定必須重啟機器人才有作用。

listen_to_pages
頁面 設定 註解說明
Commons:Village pump 
{
  "no_tail_clear": true,
  "max_title_display_width": "22em"
}
User talk:Kanashimi#Commons:Village pump/topic list
