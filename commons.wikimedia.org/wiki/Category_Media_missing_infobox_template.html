This category is used for administration or maintenance of Wikimedia Commons.

Pages missing Template:Information or other official Infobox templates. As of the end of 2014, there were ~700k files in this category, see here. Part of File metadata cleanup drive.

To add {{Information}}, one should use Magnus' add_information.php tool; this is also available as a gadget in your settings (or insert the following text with variables): 

{{Information
 | description =
 | date =
 | source =
 | author =
 | permission =
 | other fields = 
 | other versions =
}}

See Commons:Infobox templates for other infoboxes which might be better. After adding infobox template, please remove the file from this category. Use this PetScan query to find files that can be removed from this category. Use this PetScan query to find new uploads missing infobox templates. This category contains 317,611 files, which is 0.36% of all 87,585,135 files in Wikimedia Commons.
