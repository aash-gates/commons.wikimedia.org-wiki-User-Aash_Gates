This template shows that a better image should be used instead of the current image.

The template automatically sets the following categories:

• when transcluded from a SVG file: raster Bitmap version available
SVG Superseded
• when transcluded from raster files: raster PNG version available
SVG Superseded by SVG Template {{Vector version available}} should be used !

See also: Intentions

Usage

{{SupersededPNG |1= }}

The main template is Superseded. It handles each combination of superseded and superseding files.
If a redirect with wrong extension names is used, as PNG available but the superseding file is SVG,
WWWit will work correctly but the displayed text may be misleading. See also "Intentions".

Intentions

Depending on the combination of the file where the template is used (the "origin"), and the file specified by the transclusion (the "target"), different requests are assumed.

origin file target file category meaning
vector raster Bitmap version available a bitmap version to that vector graphics is available;
it is possible to specify whether file has better quality
vector vector Superseded target supersedes origin
raster raster Superseded target supersedes origin
raster vector Superseded by SVG

(Vector version available)
 vector version available; when the better suited template
{{SupersededSVG}} is used it can be specified
which of the two files has better quality
other other Superseded target supersedes origin;
(even if video or audio files, the template always speaks about "images")

It does not matter which of the many redirects for "Superseded" or "Bitmap version" is used, after checking the two file extensions the template decides about categorization.
Even when SupersededJPG (or SupersededPNG) is used for another file extension (bitmap or vector graphic, or audio/video file) it will work well –
but it seems not very wise to specify e.g. Bitmap version available when the target file is a vector version (nevertheless it's seducing to use always the short {{bva}}).
With the parameter "reason" the relation of the two files can be specified, and other information may be given.

Localization

This template is localized through {{Autotranslate}}.

To add your language to the list of languages supported by this template, please copy the code of {{Bitmap version available/en}} resp. {{Superseded/en}} (or any other language version you prefer to translate) and replace the text strings in there. Please change the parameter lang from en (or whatever language you are translating) to the language code of your language:
