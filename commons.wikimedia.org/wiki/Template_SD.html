Usage

{{SD |1= |2= }}

All pages
G1 if test page, accidental creation, or page containing nonsense or no valid content
G2 if unused and implausible, or broken redirect
G3 if content intended as vandalism, threat, or attack
G4 if recreation of content previously deleted per community consensus
G5 if temporary deletion for history cleaning or revision suppression
G6 if uncontroversial maintenance
G7 if author or uploader request deletion
G8 if page dependent on deleted or non-existent content
G9 if office action
G10 if files and pages created as advertisements
G11 if blatant text copyright violation
Galleries • Categories • Project pages • Files • Templates • User pages
GA1 if gallery without images or other media files
GA2 if user intended to create encyclopedic content
C1 if improperly named category
C2 if unuseful empty category
COM1 if improperly filled or routinely emptied deletion request or log
F1 if clear copyright violation
F2 if fair use content
F3 if derivative work of non-free content
F4 if failed license review
F5 if missing essential information
F6 if license laundering
F7 if file is empty, corrupt, or in a disallowed format
F8 if exact or scaled-down duplicate
F9 if embedded data
F10 if personal photos by non-contributors
T1 if duplicate template
T2 if unused template
U1 if user requested deletion in own user space
U2 if user page of a non-existent user
U3 if inappropriate use of user pages

TemplateData

TemplateData is a way to store information about template parameters (the description of those and of the whole template) for both humans and machines. It is used by VisualEditor and possibly other tools like Upload Wizard.

Existing template documentation
At Wikimedia Commons, it is recommended to use {{TemplateBox}} with either ‎useTemplateData=1 or ‎useTemplateData=only on the ‎/doc subpage and transcluding it with {{Documentation}} into the template. ‎<nowiki>-tags can be wrapped around the arguments, if required, to avoid templates being expanded.

Newly created template documentation and imports
Another option, especially for imported templates, or for users with JSON experience, is placing raw ‎<templatedata>-tags into the Wikitext of the template, as described in various Wikipediae.

Discussion
There is an ongoing discussion about that matter. Feel invited to comment if you are experienced in either way, your time permits and if you like to share your opinion or to make a suggestion.
