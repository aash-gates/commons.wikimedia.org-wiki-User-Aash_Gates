This template is for Wikisource, etc. which requires that images appear EXACTLY as they did in the work in question, so updated versions, corrections to errors(!), etc. can be prevented from ruining the original work with this template

This template is localized through {{Autotranslate}}. The layout of the template can be found under Template:Original/layout.

To add your language to the list of languages supported by this template, please copy the code of Template:Original/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.
