Used to request file rename, so this template is for media only.

  • For categories use {{Move}}. For any other pages, use the "move" tab (Special:MovePage/FULLPAGENAME) or {{Edit request}}.
  • If you have already uploaded the same file under a new name, use {{Duplicate|the new name}} instead of {{Rename}}.

The template automatically sets the following categories:

  • Category:Media requiring renaming, if at least the first parameter is given.
  • Category:Incomplete media renaming requests, if the request is incomplete.
  • Category:Media renaming requests needing target, if without the first or any parameter.

Usage

{{Rename |1= |2= |3= |user= }}

Named parameter (3= or reason=) must be used when an equal sign appears somewhere in the text of the reason given (as for example, inside a link).

TemplateData

TemplateData is a way to store information about template parameters (the description of those and of the whole template) for both humans and machines. It is used by VisualEditor and possibly other tools like Upload Wizard.

Existing template documentation
At Wikimedia Commons, it is recommended to use {{TemplateBox}} with either ‎useTemplateData=1 or ‎useTemplateData=only on the ‎/doc subpage and transcluding it with {{Documentation}} into the template. ‎<nowiki>-tags can be wrapped around the arguments, if required, to avoid templates being expanded.

Newly created template documentation and imports
Another option, especially for imported templates, or for users with JSON experience, is placing raw ‎<templatedata>-tags into the Wikitext of the template, as described in various Wikipediae.

Discussion
There is an ongoing discussion about that matter. Feel invited to comment if you are experienced in either way, your time permits and if you like to share your opinion or to make a suggestion.

This file has been tagged for renaming by Jeff G. (talk · contribs · count · global contribs)

You can help by suggesting a new name. To do this use the template like this:

{{rename|new name.jpg|{{{8}}}}}

This media object can not yet be renamed because:

  • the template {{Rename}} does not specify a target media name or
  • the requested file extension is different from the current file extension or
  • a file already exists at the target media name.

Please help fix this by:

  • checking if the rename meets one of the valid reasons on Commons:File renaming:
  • removing this template if a rename is not appropriate or
  • replacing this template with {{Rename|target name|numeric reason|text reason for rename}}

Ensure the target has the same extension, and that there is no image there.

Admins can also do the rename now by

  1. check if file is used and
  2. add {{Universal replace|Rename|newimage|reason=[[Commons:File renaming|File renamed]]}} to CommonsDelinker’s commands (Quick adding) (remember to replace "newimage" with the actual name)
  3. clicking here to perform the move (don't forget to remove this template after that).
and tags the file as Category:Media renaming requests needing target.
