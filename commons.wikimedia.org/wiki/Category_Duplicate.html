This category contains files that are exact or scaled-down duplicates of other files. Files that are not exact duplicates should be requested for deletion on Commons:Deletion requests.

Redirect the deleted duplicate filename to the remaining file unless it is misleading or a very recent upload.

If not added by uploader please consider Commons:File renaming and/or to move page to get history moved to new file.

  • Use the Process duplicate link on the {{Duplicate}} template to easily compare files with their "duplicates".
  • See Special:ListDuplicatedFiles for search duplicates

This category is sorted by the date&time the image was last modified, so initially that will be the date the {{Duplicate}} template was added. So oldest listed first, new entries displayed at the end of the category (if the file pages are edited again while waiting in this queue they will end up at the end of the queue again).
