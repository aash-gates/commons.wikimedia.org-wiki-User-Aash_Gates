Cropping

I have just started to use the cropping tool. In some cases I would like to use BOTH images, the cropped and the uncropped. I looked to see if I can rename a cropped image, which would solve this problem, but I didnt find that option. Thanks in advance. --Smokefoot (talk) 14:56, 23 September 2022 (UTC)Reply[reply]

@Smokefoot: When using the cropping tool, you can choose between "Overwrite" and "Upload as new file". If you choose "Upload as new file", you can enter a new filename for the cropped image and it will be saved separately from the original. —Granger (talk  · contribs) 21:28, 23 September 2022 (UTC)Reply[reply]
@Granger: A belated thank you for instructions. Very helpful.--Smokefoot (talk) 14:17, 29 September 2022 (UTC)Reply[reply]
