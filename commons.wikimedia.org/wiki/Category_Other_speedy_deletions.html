This category contains various candidates for speedy deletion.
  • Before adding or deleting any files here, you must understand the criteria for speedy deletion.
  • If a candidate does not meet the criteria for speedy deletion, replace the tag with a regular deletion request.
  • Please do not delete redirects. Other projects, including those using InstantCommons, might be using the file even though they don't show up in the global usage. Deleting the redirects would break their file reference.
  • Check whether a file has been previously uploaded to Wikimedia Commons.
  • The Bad Old Ones tool can be used to assist deletion.
