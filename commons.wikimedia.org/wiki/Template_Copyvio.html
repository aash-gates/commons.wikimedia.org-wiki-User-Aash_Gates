This file may meet the criteria for speedy deletion.

This file is a copyright violation because it is copyrighted and not published under a free license.
The file is subject to speedy deletion unless it is relicensed according to the Commons licensing policy.

Click here to show further instructions

Usage: Please explain why the file is a copyright violation by typing {{copyvio|1=Reason}}. If you have a URL that shows that the file is a copyright violation, you can use {{copyvio|source=URL}}. This template should only be used for obvious cases. For non-obvious cases, use {{Delete}} and follow the instructions.

Please notify the uploader by adding the following to their talk page:

{{subst:copyvionote|1=Template:Copyvio}} ~~~~

For multiple files from one user, explaining copyrights might be more useful.

Usage

{{Copyvio |1= |source= }}

Or use the shorthand {{db-f9|… parameters as described …}}

This template should only be used for obvious cases. For non-obvious cases like freedom of panorama or threshold of originality-related deletions, use {{Delete}} and follow the instructions.

TemplateData

TemplateData is a way to store information about template parameters (the description of those and of the whole template) for both humans and machines. It is used by VisualEditor and possibly other tools like Upload Wizard.

Existing template documentation
At Wikimedia Commons, it is recommended to use {{TemplateBox}} with either ‎useTemplateData=1 or ‎useTemplateData=only on the ‎/doc subpage and transcluding it with {{Documentation}} into the template. ‎<nowiki>-tags can be wrapped around the arguments, if required, to avoid templates being expanded.

Newly created template documentation and imports
Another option, especially for imported templates, or for users with JSON experience, is placing raw ‎<templatedata>-tags into the Wikitext of the template, as described in various Wikipediae.

Discussion
There is an ongoing discussion about that matter. Feel invited to comment if you are experienced in either way, your time permits and if you like to share your opinion or to make a suggestion.
