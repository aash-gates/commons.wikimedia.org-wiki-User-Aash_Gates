Use {{No license since}} by adding {{subst:nld}} to categorize files here.

See also category: Media uploaded without a license.– those should (usually are already) be tagged with {{subst:nld}} to be sorted in here, too.

  • Use Petscan to find files tagged with {{No license since}} template, that have a license (which might or might not be valid).
  • Use Petscanto find files uploaded in last 7 days without a license.
    • If output from PetScan is in "gallery" format than it can be copied to some personal sandbox page and VisualFileChange can be used to do batch changes.
  • 7 days ago eligible for deletion: 7 October 2022
  • purge this page's cache
